Three.js solar model.

## Technology stack:

-   Vanilla JS + Three.js
-   Vanilla CSS

## Features:

-   Includes models for Sun, All 9 Planets (including Pluto), Moon and Asteroid belt.
-   Interactive Orbit controls.
-   Brief description about each model with voice over.
